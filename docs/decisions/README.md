# Decision Log Index

ADR format: short, factual, and immutable once accepted.

- `ADR-0001-determinism-default.md`

- `ADR-0003-explainability-as-product-surface.md`
- `ADR-0004-no-silent-degradation.md`

Conventions:
- one decision per file
- include alternatives considered
- include rejected alternatives
- include operational consequences
